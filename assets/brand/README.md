# Triage Kit Brand Assets

## TK Hex Mark

Primary transparent mark:

- `tk-hex-mark.png` - 1254px transparent master

Prepared icon sizes:

- `icons/tk-hex-512.png`
- `icons/tk-hex-256.png`
- `icons/tk-hex-128.png`
- `icons/tk-hex-64.png`
- `icons/tk-hex-32.png`
- `icons/tk-hex-16.png`

Use the 32px or larger versions in application headers, documentation, and release materials. The 16px version is intended only for favicon-style use.

The generated chroma-key source is retained under `source/` for future editing. Do not use the source image directly in the application.
